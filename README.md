# random_bot

# Чем бот занимается
Бот занимается созданеим рандомных чисел от 1 до 100

# Какие есть команды в боте 
В боте есть команды 
/start - перезапускает бота 
/random - создает рнадомное число от 1 до 100
/help - показывает все доступные команды 
/progess - прогресс комапании VECTORBOT

# Какие есть особенность в этом боте 
Главная команда это /progress потому что после ввода команды бот напишет вот это: Мы научились делать команды, если отправите фото тогада он скажет какое красивое фото, тригер на определенное слово напишите слово привет то он напшите привет и ваш user_name если  id то бот напишет ваше айди.
